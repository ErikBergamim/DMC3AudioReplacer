# DMC3 Audio Replacer

.NET 3.5 program that automates the conversion of WAV files for insertion into Devil May Cry 3 from HD Collection

## Credits

- Tool for converting the audios: [MFAUDIO](https://gamebanana.com/tools/6656)

# Tutorial

Download and install 7Zip
<img width="720" height="255" alt="1" src="https://github.com/user-attachments/assets/c1fe474f-2f74-40f4-9f55-1e71618a2e5f" />

Download and extract the latest release
<img width="685" height="412" alt="2" src="https://github.com/user-attachments/assets/938351eb-900a-4037-a50f-2ef69929b688" />

Go to your DMC3 folder, and extract dmc3-0.nbz file using 7Zip
<img width="750" height="795" alt="3" src="https://github.com/user-attachments/assets/36e45734-081a-483d-958c-ddec1511658b" />

Open the sound manager and browse for the GData.afs folder you just extracted
<img width="1010" height="670" alt="4" src="https://github.com/user-attachments/assets/41492328-176f-4288-8091-3585216b684f" />

Mod your files, then turn GData.afs back to a zip file, and rename it to dmc3-0.nbz, and replace it back to dmc3 folder
<img width="999" height="714" alt="5" src="https://github.com/user-attachments/assets/239e24b3-62eb-4936-9571-d5145476ff79" />

Now test the sounds in game
